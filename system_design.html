Feature 02
Personalized Diet Planning & Nutrition
3-stage organ-aware nutrition pipeline with 160+ clinical recipes
3-Stage ML Pipeline
1
Stage 1 — Organ Risk Estimation (XGBoost)
Predicts low/medium/high risk for liver, kidney, gut, and nerve. Isotonic calibration ensures reliable probabilities.
2
Stage 2 — Nutrition Planning (LightGBM)
Multi-output regressor predicts calorie, protein, carbs, fats, hydration targets. Clinical safety layer enforces hard constraints (e.g., alcohol users always get liver focus).
3
Stage 3 — Meal Ranking (LightGBM LambdaRank)
Selects 5-7 optimal meals/day from 160+ clinically-tagged recipes (Indian, Western, Pan-Asian) timed for breakfast, lunch, dinner, snacks.
4
Feature Engineering
substance_burden_score, liver_stress_index, gut_disruption_index, and 12+ derived features
XGBoost LightGBM LambdaRank Isotonic Cal. Pydantic pandas

🗓️
Feature 03
Recovery Engine — 7-Day Activity Planning
4-model clinical pipeline generating week-long personalized schedules
4-Model ML Pipeline
1
Model 1 — Risk Estimator (XGBoost Multi-Output)
Classifies 6 ordinal targets: addiction_risk, weakness, numbness, recovery_intensity_need, sleep_risk, stress_risk (classes 0-4)
2
Model 2 — Organ Stress Prioritizer (XGBoost)
Ranks recovery priority for liver, kidney, gut, nerve (0=low → 3=critical). Substance-specific safety floor rules override predictions.
3
Model 3 — Activity Planner (XGBoost, 8 targets)
Predicts activities_per_day, walk/breathing/mindfulness durations, craving sessions, sleep prep, intensity band, rest breaks
4
Clinical Validator — Zero-Violation Safety Layer
Enforces 100+ curated WHO/NIDA/SAMHSA activities, substance-specific tagging, contraindication warnings, clinician referral flag if confidence < 0.8
XGBoost Multi-Output Isotonic Cal. Rule Engine Pydantic WHO/NIDA

Feature 04
Emergency Coping Engine
Real-time 5-model crisis intervention with LSTM craving forecasting
5-Model Real-Time Pipeline
1
Model 1 — Risk Detection (XGBoost)
Regressor + classifier: continuous risk_score and discrete risk_category (low/med/high/critical)
2
Model 2 — Relapse Prediction (XGBoost)
Binary classifier: predicts relapse_within_24h probability
3
Model 3 — Intervention Recommender (LightGBM)
Recommends top-3 personalized interventions from library (breathing, meditation, walk, etc.)
4
Model 4 — Contextual Bandit (RL)
Q-learning style personalization: learns from user feedback which interventions work best per user
5
Model 5 — Craving LSTM (PyTorch)
Predicts hours until next craving episode from 8 sequential features — enables proactive intervention
XGBoost LightGBM LSTM Contextual Bandit Q-Learning PyTorch Pydantic

Design Principles

Core values built into every model and pipeline decision

🔒
Privacy-First
PII stripped before any model sees data. No user_id stored in artifacts.
🏥
Clinical Safety
Hard constraint overrides ensure 0% safety violations. WHO/NIDA guidelines embedded.
💡
Explainable AI
SHAP values on every prediction. Top-3 feature contributions surfaced to users.
⚡
Real-Time Ready
FastAPI endpoints, joblib artifacts, <200ms inference for emergency response.
🧬
Multi-Modal ML
Classification, regression, survival, clustering, anomaly, LSTM, and RL — all in one suite.
🎯
Holistic Support
Addresses risk, nutrition, physical activity, mental health, and crisis — simultaneously.
